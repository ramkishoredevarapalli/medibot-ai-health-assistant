MediBot is an AI-powered multilingual health assistant 
built as a Telegram bot. Users can type their symptoms 
in any Indian language — English, Hindi, Telugu, Hinglish 
or Tenglish — and MediBot will predict the most likely 
disease using a trained Random Forest ML model and provide 
a detailed explanation using Google Gemini 2.5 Flash AI.

Built to solve a real problem — making healthcare guidance 
accessible to every Indian, in their own language, on the 
app they already use every day.
